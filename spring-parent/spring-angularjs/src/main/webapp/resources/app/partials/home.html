Welcome to my sample application

As you can see, is a very simple stupid application in order to demostrate how it's easy to develop with the following frameworks:

  • Spring MVC
  • Angularjs

With Spring MVC I developed the server-side controllers using rest and json for data interchange. Also this data is used in the front end part, that has been developed with angularjs and the beautiful bootstrap css. I hope that it helps my simple proof of concept! :)
